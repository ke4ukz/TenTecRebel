Computer frontend for Ten Tec Rebel 506 with custom firmware by KE4UKZ

This program is designed for testing the serial control protocol and to display data output by the radio.

Requires .NET framework 2.0
Build with Visual Studio 2013
Serial port is fixed in code as COM9@115200 (can be changed before runtime)
No error checking... pretty much any error will be fatal
No help or explanations in the program, but questions will be answered promptly
